## Filter Sidebar Components

### Location-Based Filtering
- **ZIP Code Input**: Text input for postal code
- **Current Location Button**: Geolocation-based filtering
- **Radius Slider**: 10-60 mile range selector with visual feedback
- **Search Radius Display**: Shows current radius selection (35 miles default)

### Store Location List
- **Location Items**: Checkboxes with store name and product count
- **Search Functionality**: "Search Location" input for filtering stores
- **Hierarchical Display**: City, address, and item count format
- **Scrollable List**: Long list of locations with consistent formatting

### Category Filtering
- **Category Checkboxes**: Product categories with item counts
- **Hierarchical Structure**: Main categories with subcategories
- **Count Display**: Shows available items per category

### Price Range Filter
- **Dual Range Slider**: Min/max price selection
- **Price Display**: Shows current range ($0.00 - $29,000.00)
- **Real-time Updates**: Slider values update display immediately

### Shoe Size Filter
- **Size Grid**: Comprehensive size options including:
  - Adult sizes (1-18+)
  - Youth sizes (1Y-13Y)
  - Child sizes (1C-13.5C)
  - Width options (W, M, B, D, EE)
- **Search Functionality**: "Search Shoe Size" input
- **Count Display**: Shows available items per size

### State Filter
- **State Checkboxes**: Geographic filtering by state
- **Search Functionality**: "Search States" input
- **Item Counts**: Shows inventory per state
