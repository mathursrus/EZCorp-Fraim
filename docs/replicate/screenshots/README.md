# Screenshots Directory

This directory contains organized screenshots captured during the replicate-discovery workflow.

## Directory Structure
- `authentication/` - Login, account, and user management screens
- `navigation/` - Header, menu, and navigation elements
- `forms/` - Input forms, filters, and interactive elements
- `lists/` - Product listings, category views, and search results
- `details/` - Individual product pages and detailed views
- `mobile/` - Mobile-responsive views and interactions
- `states/` - Different application states (loading, error, empty)

## Screenshot Index
Screenshots are named descriptively and include:
- Timestamp for reference
- Page/section identifier
- State or interaction captured
